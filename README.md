# E-Commerce-Store-Analysis
A data analytics project that explores sales performance, customer behavior, and product trends in an online retail environment. The dashboard highlights key KPIs such as revenue, profit margin, payment preferences, shipping modes, and category performance, helping businesses make smarter, data‑driven decisions
